# Discord Gradient Preview

一個單一 HTML 頁面 (`discord_name_gradient_preview.html`)，用來即時預覽 HEX 漸層動畫文字，方便在 Discord 等平台使用前先調整顏色與內容。

## 使用方式

- 直接在瀏覽器中開啟 `discord_name_gradient_preview.html`。
- 在「起始 HEX」與「結束 HEX」欄位調整顏色（可用色碼或色票）。
- 在「測試文字」欄位輸入要預覽的文字，即可看到上方的漸層動畫效果。

## 語系說明

- 若使用者瀏覽器語系為英文（例如 `en-US`、`en-GB`），介面會自動切換為英文標籤。
- 其他情況預設為繁體中文介面。
